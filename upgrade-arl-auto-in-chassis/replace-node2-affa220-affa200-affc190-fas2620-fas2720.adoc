= Substitua o módulo controlador node2 AFF A220, AFF A200, AFF C190, FAS2620 ou FAS2720
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Substitua o módulo controlador node2 AFF A220, AFF A200, AFF C190, FAS2620 ou FAS2720 por um módulo controlador ASA A150, AFF A150 ou FAS2820.

Nesta etapa, o node2 está inativo e todos os dados são fornecidos pelo node1. Você deve ter cuidado para remover apenas o módulo controlador do node2. Normalmente, o node2 é o controlador B localizado no lado direito do chassi quando se olha para os controladores pela parte traseira do sistema. A etiqueta do controlador está localizada no chassi diretamente acima do módulo controlador.


CAUTION: Não desligue o chassi porque o node1 e o node2 estão no mesmo chassi e conetados às mesmas fontes de alimentação.

.Antes de começar
Se você ainda não está aterrado, aterre-se corretamente.



== Etapa 1: remova o módulo controlador AFF A220, AFF A200, AFF C190, FAS2620 ou FAS2720

Para acessar os componentes dentro do controlador node2, remova o módulo do controlador do sistema e depois remova a tampa do módulo do controlador.

.Passos
. Solte o gancho e a alça de loop que prendem os cabos ao dispositivo de gerenciamento de cabos e, em seguida, desconete os cabos do sistema e os SFPs (se necessário) do módulo do controlador, mantendo o controle de onde os cabos estavam conetados.
+
Deixe os cabos no dispositivo de gerenciamento de cabos para que, ao reinstalar o dispositivo de gerenciamento de cabos, os cabos sejam organizados.

. Retire e reserve os dispositivos de gerenciamento de cabos dos lados esquerdo e direito do módulo do controlador.
+
image:drw_25xx_cable_management_arm.png["Remoção do ARM de gerenciamento de cabos"]

. Aperte o trinco na pega do excêntrico até que este se solte, abra totalmente o manípulo do excêntrico para libertar o módulo do controlador do plano médio e, em seguida, utilizando duas mãos, puxe o módulo do controlador para fora do chassis.
+
image:drw_2240_x_opening_cam_latch.png["Abrir o manípulo do excêntrico"]

. Vire o módulo do controlador ao contrário e coloque-o numa superfície plana e estável.




== Etapa 2: instale o módulo controlador ASA A150, AFF A150 ou FAS2820

Instale o módulo de controlador ASA A150, AFF A150 ou FAS2820 que você recebeu para o upgrade no nó2.

.Antes de começar
* Se você ainda não está aterrado, aterre-se corretamente.
* Desconete todos os cabos, incluindo console, gerenciamento, armazenamento SAS e cabos de rede de dados, do controlador que está sendo removido.


.Passos
. Alinhe a extremidade do módulo do controlador com o compartimento B no chassis e, em seguida, empurre cuidadosamente o módulo do controlador até meio do sistema.
+

TIP: O compartimento B está localizado no lado direito do chassi.

+

WARNING: Não introduza completamente o módulo do controlador no chassis até ser instruído a fazê-lo mais tarde no procedimento.

. Cable as portas de gerenciamento e console para o módulo do controlador node2.
+

NOTE: Como o chassi já está ligado, o nó 2 inicia a inicialização assim que estiver totalmente encaixado.  Para evitar que o nó 2 seja inicializado, conecte os cabos de console e de gerenciamento antes de inserir completamente o módulo controlador.

. Empurre firmemente o módulo do controlador para dentro do chassi até que ele atenda ao plano médio e esteja totalmente assentado.
+
O trinco de bloqueio sobe quando o módulo do controlador está totalmente assente.

+

NOTE: Para evitar danificar os conetores, não utilize força excessiva ao deslizar o módulo do controlador para o chassis.

. Conecte o console serial assim que o módulo estiver encaixado e esteja pronto para interromper o AUTOBOOT do node2.
. Após interromper o AUTOBOOT, o node2 para no prompt do LOADER. Se você não interromper o AUTOBOOT a tempo e o node2 começar a inicializar, aguarde o prompt e pressione Ctrl-C para entrar no menu de inicialização. Depois que o node parar no menu de inicialização, use a opção `8` para reinicializar o node e interrompa o AUTOBOOT durante a reinicialização.


.O que se segue?
Após instalar o módulo controlador ASA A150, AFF A150 ou FAS2820, link:netboot-node2.html["netboot node2"].
