= Aposentar-se node2
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Para descontinuar o node2, desligue-o corretamente e remova-o do rack ou chassi.

.Passos
. Retomar a operação:
+
`system controller replace resume`

+
O nó pára automaticamente.

+
.. Opcionalmente, desative o node2 após a conclusão da atualização. Consulte link:decommission_old_system.html["Desativar o sistema antigo"].




.O que se segue?
Após a desativação do node2, link:install_boot_node4.html["instale e inicialize o node4"].
